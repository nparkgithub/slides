Why VLA Models Are Needed

Traditional robot control required training a new policy for every single task.

GR00T aims to generalize across many tasks and many robot bodies using one foundation model.

Traditional Approach

Policy for Task A
Policy for Task B
Policy for Task C

Trained separately per task → doesn't scale

Foundation Model Approach

GR00T (one model)

Change only the vision input and language instruction to perform different tasks; adapts to new robots with light fine-tuning

The Core Idea: Dual-System Architecture

Inspired by dual-process theories of human cognition (Kahneman's System 1 / System 2).

System 2
Vision-language model
Slow, deliberate thinking
→
System 1
Diffusion transformer

System 2 understands the scene and plans; System 1 generates joint commands in real time.

Fine-Tuning & Deployment Workflow

  1. Prepare data — collect robot demonstrations (video, state, action) in LeRobot format
  2. Zero-shot inference — validate the base model on pretrained embodiments
  3. Fine-tune — adapt to your own robot and modality configuration
  4. Evaluate — open-loop evaluation, then simulation or real hardware testing
  5. Deploy — connect via the policy API to your robot controller, accelerate with TensorRT

Real-World Case: Unitree G1

GR00T supports whole-body humanoid control through the UNITREE_G1_SONIC embodiment tag and the GEAR-SONIC controller.

VR teleoperation for demo collection → VLA fine-tuning → real-time deployment via PolicyServer + SONIC decoder, all in one connected pipeline.

A single policy performs language-conditioned manipulation and locomotion together, end to end.

Key Takeaways

  • GR00T uses a dual architecture: System 2 (VLM) + System 1 (Diffusion Transformer)
  • Flow-matching generates continuous actions from noise in near real time
  • Embodiment-conditioned action heads let one model support multiple robot bodies
  • The data pyramid (web → simulation → real robot) drives generalization
  • Fine-tuning and deployment are supported end to end on real humanoids like the Unitree G1
